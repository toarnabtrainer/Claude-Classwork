---

## 🔶 Important Artifact Update

For classroom reliability, learners should create Artifacts using:

```text
Claude sidebar ➜ Artifacts ➜ New Artifact
```

Alternative method:

```text
New chat or Project chat ➜ ask Claude to create an Artifact
```

The alternative may work, but it is less predictable during a live class.

## ✨ Learning Outcomes

Learners should be able to:

```text
☑️ Create a Claude Project

☑️ Add project instructions
☑️ Test grounded answers

☑️ Create an Artifact using Artifacts ➜ New Artifact
```
