# ☸️ Recommended Topic Sequence

This file gives the recommended order for the complete Claude Training session.

---

## 🔰 Main Course Sequence

| Order | Topic | Recommended Module | Suggested Time |
|---:|---|---|---:|
| 1 | Claude fundamentals | Module 1 | 30–40 min |
| 2 | Prompt engineering | Module 1 | 45–60 min |
| 3 | Productivity workflows | Module 2 | 45–60 min |
| 4 | Projects and knowledge management | Module 3 | 45–60 min |
| 5 | Artifacts | Module 3 / Module 7 | 30–45 min |
| 6 | Documents, PDFs, images, and analysis | Module 4 | 45–60 min |
| 7 | Evaluation, safety, and privacy | Module 5 | 45–60 min |
| 8 | Developer and automation concepts | Module 6 | 60–90 min |
| 9 | Add-on feature lab | Module 7 | 90–150 min |

---

## ✨ Best Order for Feature Demonstrations

For Projects, Artifacts, Design, Skills, and Connectors, use this sequence:

```text
Projects ➜ Artifacts ➜ Design ➜ Skills ➜ Connectors
```

Learners should understand the logic:

```text
Projects = workspace and knowledge
Artifacts = reusable output
Design = visual output
Skills = reusable behavior
Connectors = external data and tools
```

---

## 🔶 Add-on Module Recommended Order

Inside Module 7, use:

```text
1. Artifacts
2. Design
3. Skills
4. Connectors
```

This is easier because learners first see outputs, then visual creation, then reusable behavior, and finally external integrations.
