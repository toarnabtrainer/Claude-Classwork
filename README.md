# 🔰 Claude Training Course — Simple GitHub Edition

Welcome to the **Claude Training Course**. This repository is designed for classroom delivery, self-learning, and GitHub-based sharing.

The course contains:

```text
✅ 6 main modules
✨ 1 optional add-on feature lab
```

Learners should understand that this course is intentionally simplified. Each module keeps only the files needed for smooth delivery and practice.

## 🔶 Important Feature Notes

### Projects

Learners should understand that Projects are organized workspaces with context, chats, instructions, and project knowledge.

### Artifacts

Learners should understand that Artifacts can be created in two ways:

```text
✅ Reliable classroom method:
Claude sidebar ➜ Artifacts ➜ New Artifact

✳️ Alternative method:
New chat ➜ ask Claude to create a substantial standalone output
```

For classroom reliability, use:

```text
Artifacts ➜ New Artifact
```

### Skills

Learners should understand that Skills are reusable instruction packages. A custom Skill is normally uploaded as a ZIP file containing a Skill folder and a `Skill.md` file.

### Design

Learners should understand that Claude Design is used for visual outputs such as designs, prototypes, presentations, one-pagers, and design-oriented layouts.

### Connectors

Learners should understand that Connectors let Claude work with external apps and data sources, subject to permissions and account availability.

## 🧪 MCQ Format

MCQs use GitHub-compatible click-to-reveal answers:

```markdown
<details>
<summary>✅ Show answer</summary>

Correct answer appears here.

</details>
```

This lets learners test themselves without seeing the answer immediately.

---

## 🔴 Responsible Use Reminder

Learners should understand:

```text
❌ Do not paste private, sensitive, or regulated information unless allowed.
❌ Do not accept AI outputs blindly.
✅ Verify important facts.
✅ Use human review for legal, financial, medical, HR, compliance, and safety-related work.
✅ Check permissions before using Connectors.
```
