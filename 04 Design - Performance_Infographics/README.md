# Performance Infographics

## Project overview

**Performance Infographics** is a learner-friendly Claude Design project for converting corporate performance data into clear, professional, and decision-focused visual communication.

The project uses one fictional quarterly business case so that the design can be created, tested, and demonstrated without using confidential company information. The final output is produced inside Claude Design by following a sequence of complete prompts.

The project is designed for:

- business analysts;
- managers and team leaders;
- strategy and finance professionals;
- learning and development teams;
- employees who need to explain performance visually;
- users who want to practise Claude Design with a realistic corporate example.

---

## What you will create

The primary output is a **single-page landscape performance infographic** for the fictional company **NorthBridge Business Services Ltd.**

The infographic will communicate:

1. executive performance summary;
2. six key performance indicators;
3. revenue, margin, and operating-profit trends;
4. customer and operational performance;
5. regional and business-unit highlights;
6. strategic-initiative progress;
7. enterprise risks;
8. management priorities and recommended actions.

An optional prompt is also included for creating a portrait version after the landscape design is complete.

## Recommended setup

### Step 1 — Review the project

Read this README first. Then review:

```text
01_Design_Blueprint_and_Style_Guide.md
```

This will help you understand the intended design before starting.

### Step 2 — Prepare the source content

Upload or make available in the same Claude workspace:

```text
01_Design_Blueprint_and_Style_Guide.md
02_Sample_Performance_Content_and_Data.md
```

Keep `03_Step_by_Step_Design_Prompts.md` open for copying prompts. You may also upload it as a reference file, but run only one build prompt at a time.

### Step 3 — Start the design

Open Claude Design and begin a new design conversation. Run **Prompt 1** from:

```text
03_Step_by_Step_Design_Prompts.md
```

Ask Claude to create the initial landscape infographic.

### Step 4 — Build progressively

Continue through the prompts in order. Review the design after each major stage before moving to the next prompt.

Do not paste every prompt at once. Progressive refinement makes it easier to understand what each instruction changes.

### Step 5 — Validate before finalising

Use the numerical-audit, readability, and final-quality prompts near the end of the prompt file. Check the infographic against the source content before using it in a demonstration.

## Expected final output

The completed landscape infographic should:

- fit on one page;
- use a clear 16:9 corporate layout;
- present the main conclusion before detailed data;
- display six executive KPIs with actual, target, and status;
- use no more than three primary charts;
- show strategic initiatives and enterprise risks clearly;
- contain concise management takeaways;
- distinguish positive, caution, and adverse performance consistently;
- remain readable when projected during a meeting;
- use only the supplied fictional data;
- contain no placeholders, unsupported claims, or decorative charts.

## Suggested colleague demonstration

A concise demonstration can be completed in five stages.

### 1. Explain the objective

State that the project converts a quarterly performance report into a decision-focused visual summary.

### 2. Show the source material

Open `02_Sample_Performance_Content_and_Data.md` and briefly show the KPI, trend, initiative, and risk information.

### 3. Show progressive creation

Run the first build prompt and then one or two refinement prompts to demonstrate how Claude Design changes hierarchy, charts, and messaging.

### 4. Test accuracy

Use the numerical-audit prompt and show how the design is checked against the source data.

### 5. Present the completed infographic

Explain the executive story:

- growth continued;
- revenue remained below target;
- margins and operating profit weakened;
- customer advocacy improved;
- retention and service reliability require attention;
- management should prioritise margin recovery, retention, and delivery discipline.

## Completion checklist

Before treating the design as complete, confirm that:

- [ ] the company name and reporting period are correct;
- [ ] all six executive KPI values match the source file;
- [ ] percentage changes and percentage-point changes are labelled correctly;
- [ ] charts use consistent periods and units;
- [ ] the main message is visible within a few seconds;
- [ ] the page is not overcrowded;
- [ ] text is readable at presentation size;
- [ ] green, amber, and red are used consistently;
- [ ] colour is not the only status signal;
- [ ] initiative progress and risk levels are understandable;
- [ ] recommended actions are specific and prioritised;
- [ ] the fictional-data note is visible;
- [ ] no confidential or real-company data has been introduced.

## Data and responsible-use note

All figures in this package are fictional and are provided only for learning and demonstration.

For real corporate use:

- replace the sample data with approved information;
- validate calculations and definitions with the responsible business owners;
- remove confidential information that should not be shared;
- ensure charts do not exaggerate changes through misleading scales;
- obtain management approval before external circulation;
- preserve a source record for every material figure.

## Project outcome

After completing this project, a learner should be able to:

- structure a performance story;
- select suitable KPIs and charts;
- create a professional corporate infographic in Claude Design;
- refine a design through conversational instructions;
- validate visual outputs against source data;
- explain the design process clearly to colleagues.
