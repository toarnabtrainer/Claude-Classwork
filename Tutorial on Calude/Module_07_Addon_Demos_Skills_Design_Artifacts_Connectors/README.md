# ✨ Module 7 — Add-on Demos: Skills, Design, Artifacts, and Connectors

## ✅ Purpose

This add-on module gives dedicated classroom demonstrations for four Claude features:

```text
1. Artifacts
2. Design
3. Skills
4. Connectors
```

The recommended order is:

```text
Artifacts ➜ Design ➜ Skills ➜ Connectors
```

Learners should understand that this add-on is a feature lab, not a replacement for the core modules.

## 🔴 Safety Reminder

Learners should understand:

```text
❌ Do not upload secrets into Skills.
❌ Do not publish Artifacts without review.
❌ Do not connect private services unnecessarily.
✅ Review permissions before using Connectors.
✅ Use fictional data for demonstrations.
```
