# ✨ Prompt Templates to Copy

## 1. CRAFT Prompt Template

```text
Context:
[Explain the background]

Role:
Act as [expert role].

Action:
Your task is to [specific task].

Format:
Return the answer as [table / checklist / report / JSON / email].

Tone:
Use [simple / professional / friendly / concise] tone.

Quality rules:
- State assumptions clearly.
- Do not invent facts.
- Ask clarifying questions only if necessary.
```

---

## 2. Explanation Prompt

```text
Teach me [TOPIC] as if I am a beginner.

Include:
1. Simple explanation
2. Real-world example
3. Common mistakes
4. Practice exercise
5. Short quiz
```

---

## 3. Business Analysis Prompt

```text
Act as a business analyst.

Analyze the following situation:
[PASTE SITUATION]

Return:
1. Summary
2. Key problems
3. Risks
4. Opportunities
5. Recommended next steps
```

---

## 4. Output Improvement Prompt

```text
Review this output:
[PASTE OUTPUT]

Improve it for:
1. Accuracy
2. Structure
3. Clarity
4. Practical usefulness
5. Audience fit

Return the improved version.
```

---

## 5. Verification Prompt

```text
Check the following answer for possible unsupported claims.

Return:
1. Claims that are clearly supported
2. Claims that need verification
3. Missing information
4. Safer revised wording
```
