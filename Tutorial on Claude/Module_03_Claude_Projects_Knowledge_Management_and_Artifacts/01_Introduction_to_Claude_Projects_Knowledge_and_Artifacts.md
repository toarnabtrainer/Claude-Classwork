# 🔰 Introduction — Claude Projects, Knowledge Management, and Artifacts

## 1. What Learners Should Understand

Claude Projects help organize work around a specific purpose.

A Project can include:

```text
✅ Project-specific chats
✅ Uploaded knowledge files
✅ Project instructions
✅ Reusable workflows
✅ Context for a specific business, team, or topic
```

---

## 2. Simple Explanation

```text
Project = workspace
Project Knowledge = uploaded source material
Project Instructions = rules and role for Claude
Artifact = reusable output
```

---

## 3. Why Projects Matter

Learners should understand that a normal chat is useful for one-off questions, but a Project is better when work needs:

```text
☑️ Repeated context
☑️ Uploaded knowledge
☑️ Consistent tone
☑️ Policy-based answers
☑️ Team or business workflows
```

---

## 4. What to Upload Into Project Knowledge

Good files:

```text
✅ Product overview
✅ SOP
✅ FAQ
✅ Policy document
✅ Tone guide
✅ Project brief
✅ Workflow guide
```

Avoid:

```text
❌ Unrelated files
❌ Old or conflicting documents
❌ Sensitive data not needed for the task
❌ Private credentials
```

---

## 5. Project Instructions

Project instructions tell Claude:

```text
☑️ What role to follow
☑️ What sources to use
☑️ What tone to use
☑️ What not to invent
☑️ When to escalate
☑️ What output format to use
```

---

## 6. Artifact Basics

Learners should understand that Artifacts are reusable outputs such as:

```text
✨ FAQ guides
✨ Dashboards
✨ HTML pages
✨ Reports
✨ Checklists
✨ Mini-apps
✨ Templates
```

---

## 7. Correct Artifact Creation Method for Class

Use this reliable method:

```text
Claude sidebar ➜ Artifacts ➜ New Artifact
```

Then paste or upload the source material and provide the creation prompt.

Alternative method:

```text
New chat or Project chat ➜ ask Claude to create an Artifact
```

This may create an Artifact automatically, but it is less predictable.

---

## 8. Deleting Artifacts

Learners should understand:

```text
To remove an Artifact created from a chat, delete the entire associated chat from chat history.
```

Important caution:

```text
🔴 If an Artifact was published or shared, use the visible Unpublish or Unshare option where available before assuming the shared version is no longer accessible.
```
