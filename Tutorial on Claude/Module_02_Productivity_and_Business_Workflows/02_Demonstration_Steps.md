# ✳️ Demonstration Steps — Productivity and Business Workflows

## Demo Goal

Learners should understand how Claude can transform workplace inputs into structured outputs.

---

## Part A — Email Drafting

Paste this prompt into Claude:

```text
Rewrite this message in a professional and polite tone.

Original message:
I sent the report yesterday. Please check it quickly and reply because we are already late.

Make it:
1. Polite
2. Clear
3. Action-oriented
4. Suitable for a manager
```

Learners should observe how Claude improves tone while preserving meaning.

---

## Part B — Meeting Notes

Paste this prompt:

```text
Turn the following meeting notes into structured meeting minutes.

Notes:
- Website launch delayed
- Design team needs 2 more days
- Content is almost complete
- QA testing not started
- Client asked for status by Friday
- Raj will check design
- Meena will finalize content
- QA team starts testing Thursday

Return:
1. Meeting summary
2. Decisions
3. Action items with owner
4. Risks
5. Follow-up email
```

Learners should understand how messy notes become professional output.

---

## Part C — Decision Matrix

Paste:

```text
Help me compare three options for launching a training program:

Option A: Live weekend batch
Option B: Recorded self-paced course
Option C: Corporate workshop

Compare using:
- Cost
- Time required
- Revenue potential
- Ease of delivery
- Risk

Return a table and final recommendation.
```

Learners should observe that Claude can help structure decision-making.

---

## Part D — Workflow Creation

Paste:

```text
Create a reusable workflow prompt for handling customer complaints.

The workflow should produce:
1. Complaint summary
2. Customer emotion
3. Issue category
4. Urgency level
5. Draft response
6. Escalation recommendation
```

Learners should understand the value of reusable prompts.
