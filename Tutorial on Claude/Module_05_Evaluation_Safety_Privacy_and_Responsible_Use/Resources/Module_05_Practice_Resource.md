# 🔶 Module 5 Practice Resource — Safety Review Cases

## Case 1 — Unsupported Claim

AI output:

```text
EcoServe guarantees that all customers will be fully compliant with environmental laws after using the Pro plan.
```

Review focus:

```text
- Unsupported guarantee
- Legal/compliance risk
- Safer wording
```

---

## Case 2 — Privacy Risk

AI output:

```text
Customer Rahul Verma, phone 9876543210, card number 4444 3333 2222 1111, requested refund support.
```

Review focus:

```text
- Personal data
- Payment data
- Anonymization
```

---

## Case 3 — Connector Risk

AI output:

```text
I will create a Google Calendar event tomorrow at 3 PM.
```

Missing details:

```text
- Exact date
- Time zone
- Calendar name
- Approval confirmation
- Attendees
```

---

## Case 4 — Artifact Risk

AI output:

```text
Publish this customer FAQ publicly. It includes internal escalation routes and customer email examples.
```

Review focus:

```text
- Public sharing risk
- Internal information exposure
- Review before publishing
```

---

## Case 5 — Skill Risk

AI output:

```text
This Skill includes a script that sends files to an external server.
```

Review focus:

```text
- Script safety
- Data exposure
- Need to inspect code before enabling
```

## Practice Task

Learners should evaluate each case and produce:

```text
1. Risk identified
2. Why it matters
3. Safer version
4. Human review needed? Yes/No
```
