# 🔶 Module 1 Practice Resource — BrightPath Training Scenario

## Scenario

BrightPath Learning is a fictional training company that wants to launch a beginner-friendly AI course for working professionals.

## Goal

The company wants Claude to help create:

```text
✅ Course outline
✅ Weekly learning plan
✅ Practice exercises
✅ Promotional email
✅ Learner assessment
```

## Audience

```text
Working professionals
Small business owners
Students preparing for jobs
Non-technical beginners
```

## Constraints

```text
- Keep explanations simple.
- Avoid heavy technical jargon.
- Include real workplace examples.
- Add practical exercises.
- Do not claim guaranteed job placement.
```

## Practice Task

Create a strong prompt that asks Claude to design a 2-week beginner AI training plan for BrightPath Learning.
