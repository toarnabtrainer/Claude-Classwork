# 🔰 Introduction — Documents, PDFs, Images, and Analysis

## 1. What Learners Should Understand

Claude can help analyze uploaded content such as:

```text
✅ Text documents
✅ PDFs
✅ Images
✅ Charts
✅ Tables
✅ Business notes
✅ Reports
```

---

## 2. Main Analysis Tasks

| Task | Example |
|---|---|
| Summary | Summarize a policy document |
| Extraction | Extract dates, numbers, risks, owners |
| Comparison | Compare two proposals |
| Risk review | Identify missing terms and concerns |
| Chart reading | Explain trends visible in a chart |
| Action planning | Convert findings into next steps |

---

## 3. Important Analysis Rule

Learners should ask Claude to separate:

```text
☑️ Facts found in the document
☑️ Assumptions
☑️ Missing information
☑️ Recommendations
```

---

## 4. Quote-Based Verification

For important documents, learners should ask Claude to quote or reference the source text.

Example:

```text
For every important claim, include the supporting quote or section reference.
```

---

## 5. Safety Reminder

Learners should understand:

```text
🔴 Claude can help review documents.
❌ Claude should not replace professional legal, medical, financial, HR, or compliance review.
✅ Important decisions need human verification.
```

---

## 6. Image and Chart Analysis

When analyzing images or charts, learners should ask:

```text
1. What is visible?
2. What pattern is visible?
3. What cannot be concluded?
4. What information is missing?
```

This prevents overclaiming from visuals.
