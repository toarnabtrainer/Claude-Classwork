# ✨ Document Analysis Templates to Copy

## 1. Executive Summary Template

```text
Analyze this document.

Return:
1. One-line summary
2. Executive summary
3. Key facts
4. Important dates/numbers
5. Risks
6. Missing information
7. Recommended next steps

Rules:
- Separate facts from assumptions.
- Do not invent missing details.
```

---

## 2. Quote-Based Review Template

```text
Review this document.

For each important claim, include:
1. Claim
2. Supporting quote or section reference
3. Confidence level
4. Notes or concerns
```

---

## 3. Risk Review Template

```text
Review this document for risk.

Focus on:
1. Operational risk
2. Financial risk
3. Timeline risk
4. Customer risk
5. Compliance or legal-review needs

Do not provide legal advice.
```

---

## 4. Chart Analysis Template

```text
Analyze this chart or table.

Return:
1. What is visible
2. Key trend
3. Highest/lowest values
4. Possible interpretation
5. What cannot be concluded
6. Follow-up questions
```

---

## 5. Action Plan Template

```text
Convert this document into an action plan.

Return a table with:
- Action
- Owner role
- Priority
- Due date suggestion
- Evidence/source
- Risk if delayed
```
