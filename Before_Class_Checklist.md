# ✅ Before Class Checklist

Use this checklist before delivering or attending the Claude Training session.

---

## 🔰 Account and Access

```text
☑️ Claude account is working
☑️ Correct plan access is confirmed
☑️ Internet connection is stable
☑️ Browser is updated
☑️ Claude Desktop is installed if desktop features are needed
```

---

## ✳️ Main Feature Checks

```text
☑️ New Chat opens correctly
☑️ Projects feature is visible
☑️ Artifacts space is visible, if available on the account
☑️ Design is visible, if available on the account
☑️ Customize section is visible
☑️ Skills page opens under Customize
☑️ Connectors page opens under Customize
☑️ Code or Claude Code access is checked, if needed
```

---

## ✨ Module Resource Checks

```text
☑️ Module files are extracted
☑️ Resource files are available
☑️ MCQ files render correctly on GitHub
☑️ EcoServe Project resource is ready
☑️ EcoServe Skill ZIP is ready
☑️ Add-on demo resources are ready
```

---

## 🔶 Connector Demo Checks

```text
☑️ Connector account is available, if using a live demo
☑️ Google Calendar / Drive / Gmail permission is tested, if applicable
☑️ Connector is enabled in the active chat through + ➜ Connectors
☑️ Exact date, time, and timezone are ready for calendar demos
☑️ Approval card or confirmation workflow is understood
```

---

## 🔴 Safety Checks

```text
☑️ No real customer data is used
☑️ No private employee information is used
☑️ No real passwords, API keys, or credentials are shown
☑️ Demo data is fictional
☑️ Learners understand that AI outputs need verification
```

---

## ☸️ Delivery Sequence

```text
1. Start with Topic_Sequence.md
2. Open the relevant module folder
3. Read README.md
4. Follow 01_Introduction.md
5. Perform 02_Demonstration_Steps.md
6. Use 03_Templates_To_Copy.md
7. Complete Resources activity
8. Finish with 04_MCQ_Click_to_Reveal.md
```
